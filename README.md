# Caja de Herramientas – Programas Preventivos

Sitio web estático listo para GitHub Pages.

## Archivos principales
- `index.html`: estructura.
- `styles.css`: diseño y animaciones.
- `app.js`: carrusel, navegación y ventanas de programa.
- `assets/backgrounds`: imágenes panorámicas.
- `assets/cards`: imágenes de acceso.
- `assets/logos`: logos suministrados.

## Publicar en GitHub Pages
1. Cree un repositorio nuevo.
2. Suba el contenido completo de esta carpeta a la raíz del repositorio.
3. Abra **Settings → Pages**.
4. En **Build and deployment**, seleccione **Deploy from a branch**.
5. Seleccione la rama `main` y la carpeta `/root`.
6. Guarde y espere a que GitHub publique el enlace.

## Nota
Esta versión es visual y no usa ArcGIS todavía. Cada programa abre una pantalla informativa de “Próximamente”.
